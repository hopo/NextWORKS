정규표현식을 위한 test.txt
=========================


정규표현식의 활용은
 * 테스트(검증, 찾기), 바꾸기


-- id 검증 [a-zA-Z0-9_] , 6~12
pattern = ^[a-zA-Z0-9_]{6,12}&


-- 한글이름(최소2글자)
pattern = ^[가-힇]{2,}$


-- 문자번위에 대한 약식표기
[0-9] 		 : \d , \D = [^0-9]
[a-zA-Z0-9_] : \w , \W
white space  : \s
문자경계     : \b

93년생 차정환씨는 생일은 93년 10월 11일 입니다.
김은별 840330-1434554
김민성 900319-2434449


-- ### 문자열 변경하기
-- using () 그룹화
A) 93년생 차정환씨 생일은 93년 10월 11입니다
([가-힇]{2,})\s(\d\d)(\d\d)(\d\d).+ => 4개의 그룹화

B) $2년생 $1씨는 생일은 $2년 $3월 $4일 입니다.

-- A 에서 B 로 변경된다.



-- ### file name modifier
id = "member_regist"
 -> id="member_regist"
 
id="(\w+)_(\w+)"
id="$2_$1"
